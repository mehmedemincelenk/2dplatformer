# 🎮 False Knight  
**Tema:** Kırılma  

Ortaokulda şövalye olma hayali kuran bir çocuk, zorbalar tarafından çatıdan atılır.  
Komaya girer ve zihninde bir şövalye olarak, yaratık kılığındaki zorbalarla savaşmaya başlar.  

Oyun boyunca, rüyayla gerçeğin birbirine karıştığı **“kırılmalar”** yaşar.  
Son savaşta rastgele davranışlara sahip **Big Boss** şövalyeyi yenerek karakteri komadan kurtarır.

👉 **Oyna:** [mrgezgin.itch.io/false-knight](https://mrgezgin.itch.io/false-knight)
